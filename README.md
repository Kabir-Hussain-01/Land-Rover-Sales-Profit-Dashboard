# 🚗 Range Rover Sales & Profit Dashboard (Power BI)

📁 Dataset Size: 300K+ records
📈 Domain: Automotive Sales Analytics

---

## 📊 Project Overview

Developed an interactive Power BI dashboard to analyze over **300K+ global vehicle sales records**, generating insights on revenue, profit, and sales performance.

The dashboard enables stakeholders to explore data across multiple dimensions such as **Model, Region, Fuel Type, and Transmission**, supporting data-driven decision-making.

---

## 🎯 Objectives

* Analyze sales and profit trends across regions
* Identify top-performing vehicle models
* Compare performance by fuel type and transmission
* Provide interactive and dynamic reporting

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (ETL)
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)

## ⚙️ Project Workflow

### 🔹 Data Processing (ETL)

* Cleaned and transformed **25+ data columns** using Power Query
* Improved data accuracy and consistency by approximately **20%**

### 🔹 Data Modeling

* Designed a Star Schema data model for better performance and scalability

### 🔹 DAX Measures

Created 10+ measures, including:

* Total Sales
* Revenue
* Profit
* Units Sold

Improved report performance by approximately **30%**

### 🔹 Visualization

* Built **12+ interactive visuals**
* Implemented slicers and drill-through features
* Enabled multi-dimensional analysis across:

  * Model
  * Fuel Type
  * Transmission
  * Region

Reduced manual reporting effort by approximately **40%**
